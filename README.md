Seperti biasa di node buat nyimpan data, kemudian untuk class tree nya menggunakan BST supaya ketika mengurutkan bunganya secara in-order, dia langsung terurut (di kode ini dia terurut dari kecil ke besar). Kemudian, di in-order tree dibaca dari node kiri, kemudian node utama, baru node kanan. Untuk menampilkan bunga dengan nilai terkecil, di kode ini cukup membaca node paling kiri dari BST karena di struktur BST nilai terkecil selalu berada di cabang kiri terdalam. Selanjutnya, di bagian menampilkan bunga dengan nilai genap, kodenya menelusuri seluruh node dan memeriksa apakah nilai setiap bunga habis dibagi dua, kalau iya, bunganya di tampilkan.
